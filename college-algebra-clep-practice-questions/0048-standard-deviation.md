# Standard Deviation

   - A. 5.32
   - B. 6.02
   - C. 5.16
   - D. 5.85

   - A. 14.29
   - B. 15.69
   - C. 13.79
   - D. 16.49

   - A. Data set A
   - B. Data set B
   - C. Both have the same standard deviation
   - D. Cannot be determined
